1. Дан код:

var a = 1, b = 1, c, d;
c = ++a; alert(c);           // 2
d = b++; alert(d);           // 1
c = (2+ ++a); alert(c);      // 5
d = (2+ b++); alert(d);      // 4
alert(a);                    // 3
alert(b);                    // 3

c = 2, так как сначала мы инкрементируем а, а затем присваиваем в переменную с.
d = 1, так как инкремент к b идет уже после присваивания в переменную d.
В следующих случаях аналогично.
В конце b и a равны 3, потому что до этого были дважды инкрементированы.

2. Чему будет равен x? 

var a = 2;
var x = 1 + (a *= 2);

x будет равен 5, потому что сначала производятся операции в скобках.

7. Сравнить null и 0. Объяснить результат.
Выражение 0 == null ложно, так как null - это особый тип данных, означающий
отсутствие какого-либо значения и при сравнении
на нестрогое равенство равен только самому себе.
